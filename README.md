# SCM_HW1_Reem
This repository is created to solve homework one of software configuration management  
The repository is dedicated to the first homework assignment of the SWE 442: Software Configuration Management course. The goal of this assignment is to demonstrate a practical understanding of version control, branching strategies, and change management using Git and GitHub.  
Effective SCM involves maintaining a single source of truth for all project artifacts, ensuring that every change is documented through descriptive commit messages, and using branching strategies to isolate development from the stable main branch. Additionally, regular audits and automated tracking help maintain the integrity and traceability of the software throughout its lifecycle.
